# Auckland Transport Tracker
This project is a full-stack web app that provides real-time tracking of Auckland Transport vehicles.

## 📌 Overview
Auckland Transport Tracker tracks and plots the location of all Auckland Transport vehicles, providing insight and observations in the public transport system.

## 🛠️ Technologies used
- **React**
- **Mapbox GL JS**
- **FastAPI**
- **PostgreSQL**
- **Supabase**

## 📝 Project structure
- ```database```: Not present in repository currently, a PostgreSQL database hosted via Supabase  
- ```backend```: FastAPI server that integrates Auckland Transport Developer API and manages database operations  
- ```frontend```: React web application showing all the locations of buses and other transport information
